Collage / Chance / Image

I use a flatbed scanner for image acquisition, and import multiple files into a single file. This work is designed to let me experiment with chance as an approach to aesthetic decision-making. The software I used is Photoshop.

History/Context

For this assignment I interpreted a historical event or image and re-tell it, re-enact it or modify it in a contemporary style. One image is named "Armstrong walks on moon", and the other is named "Future World " I used Photoshop to combine those two images and add my own imagination.

Silhouette/Shape/Camouflage

Monochromatic Compositions done in Illustrator. I started by choosing small, complex objects(plastic knife, spoon and fork). When placed on the scanner, they will be somewhat flat. Then, I made several changes to the element I imported from scanner. Finally, I use only those shapes to make a totally new picture.
